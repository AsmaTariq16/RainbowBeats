# 🎶 RainbowBeats  
_Bright beats for bold souls_  

A responsive **Spotify-inspired Music Player** built with **HTML, CSS, and JavaScript**.  
It allows users to browse playlists, play/pause songs, seek tracks, and adjust volume – all with a sleek and colorful UI.

## ✨ Features  

- 🎨 **Modern UI** – Colorful, gradient-based Spotify-like design.  
- 📂 **Multiple Playlists** – Predefined folders (e.g., Chill, Dark, Funky, Atif Aslam, Diljit, etc.).  
- ▶️ **Music Controls** – Play, pause, next, previous, seek, and mute/unmute.  
- 🔊 **Volume Control** – Slider + mute/unmute button.  
- 📱 **Responsive Design** – Works across desktop, tablet, and mobile.  
- 📑 **Dynamic Playlist Loader** – Loads songs from playlists dynamically via JavaScript.  

---

## 🛠️ Tech Stack  

- **HTML5** – Structure  
- **CSS3** – Styling (Flexbox, gradients, responsive media queries)  
- **JavaScript (Vanilla JS)** – Functionality (playlists, audio player, events)
